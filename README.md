# portal-chn-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相互独立，不针对任何特定域名或网站。

## 项目简介

这是一个静态 HTML 页面集合仓库。每个页面均以独立文件形式存在，可直接通过浏览器打开查看。仓库主要功能是提供一种结构化的方式存放和分享这些页面，方便访问和后续维护。

## 目录说明

```
portal-chn-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
├── index.html      # 仓库主页（可选）
└── README.md       # 本文件
```

- `pages/`：所有独立 HTML 页面存放于此，每个页面一个文件，按需命名。
- `assets/`：页面引用的样式、脚本、图片等资源，按子目录组织。
- `index.html`：可选的仓库入口页面，用于导航或展示概览。

## 页面归档说明

- 每个 HTML 页面为独立完整文档，可脱离仓库环境运行。
- 页面内容不包含任何特定商业、营销或推广信息。
- 页面间无强依赖关系，可按需单独使用。
- 归档的页面可能随仓库更新而新增、修改或移除。

## 维护说明

- 本仓库由维护者不定期更新。
- 欢迎通过 Issues 提交页面相关问题或建议。
- 如需贡献新页面，请确保其内容符合通用归档要求，无违规或敏感信息。
- 仓库不提供任何形式的服务保障，仅供存档和参考使用。
